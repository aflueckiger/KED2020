Creating, Moving, and Copying Files

mkdir data          # make a new directory

Removing Files

Watch out, there is no way back and recycle bin.

rm old.txt          # remove a file
rm -r old_data      # remove a folder with all its files

Wildcards

  • a single character: ?
  • any sequence of characters: *
mv data/*.txt new_data/.
cp **/*.txt files/.     # copy all files in a single folder
